# @eestado-utils
Componentes de utilidade do e-Estado

## Instalação

```bash
npm install @eestado-utils
```
ou
```bash
yarn add @eestado-utils
```

## Componentes

### FixReturn

```javascript
import { FixReturn } from '@eestado-utils';
```

### ModuleRoute

```javascript
import { ModuleRoute } from '@eestado-utils';
```
